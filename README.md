# Bomberman
Single and online multiplayer Bomberman game in java

## Run
Run the MySQL database with docker compose:

´´´
docker down
docker compose up -d
´´´

Give the sun bash the needed privilages:

´´´
chmod +x run.sh
´´´

Run the game with:
´´´
./run.sh
´´´
